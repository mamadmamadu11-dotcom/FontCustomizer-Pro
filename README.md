## 🌟 Overview
**FontCustomizer Pro** is a powerful Windows desktop application designed to drastically reduce your web project's footprint. Instead of loading heavy font libraries (like FontAwesome), this tool intelligently scans your project files, extracts only the icons you actually use, and generates a lightweight, custom font package.

## 💡 Why FontCustomizer Pro? (The Performance Impact)
In modern web development, loading entire icon libraries (like FontAwesome, Material Icons, or Bootstrap Icons) can severely impact your website's load time. A typical icon font file can weigh several hundred kilobytes, yet most web projects only use 10 to 50 icons.

**FontCustomizer Pro** solves this frontend bloating issue. By compiling only the exact icons your project needs, you can reduce your font file payload by up to **90%** (e.g., from 150KB down to under 15KB). 

## 📈 Boost Your SEO & Core Web Vitals
Search engines like Google heavily prioritize fast, optimized websites. Using FontCustomizer Pro directly benefits your technical SEO:
* **Faster First Contentful Paint (FCP):** Smaller font files download instantly, allowing your page text and icons to render without delay.
* **Prevent Layout Shifts & Invisible Text:** Highly compressed `.woff2` files ensure your icons load seamlessly, preventing the dreaded Flash of Invisible Text (FOIT).
* **Better PageSpeed Insights Score:** Stop getting penalized by Google Lighthouse for "Keep request counts low and transfer sizes small."

## ✨ Key Features
* **Smart Project Scan:** Automatically and recursively scans your entire project directory (HTML, C#, Python, JS, etc.) to detect used icon classes in seconds.
* **Visual Management:** Search, filter, and manually select icons using their Hex codes or names through a clean UI.
* **Auto-Generation:** Instantly generates heavily compressed `.woff2`, `.ttf`, and ready-to-use `.css` files.
* **Pro Mode (Web3 Integration):** Seamless TON Blockchain integration for secure hardware-based license activation and unlimited exports.
