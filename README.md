## 🌟 Overview
**FontCustomizer Pro** is a powerful Windows desktop application designed to drastically reduce your web project's footprint. Instead of loading heavy font libraries (like FontAwesome), this tool intelligently scans your project files, extracts only the icons you actually use, and generates a lightweight, custom font package.

## ✨ Key Features
* **Smart Project Scan:** Automatically recursively scans your entire project directory (HTML, C#, Python, JS, etc.) to detect used icon classes in seconds.
* **Visual Management:** Search, filter, and manually select icons using their Hex codes or names through a clean UI.
* **Auto-Generation:** Instantly generates heavily compressed `.woff2`, `.ttf`, and ready-to-use `.css` files.
* **Web Performance Optimization:** Reduces font file sizes by up to 90%, directly boosting your PageSpeed and Core Web Vitals.
* **Pro Mode (Web3 Integration):** Seamless TON Blockchain integration for secure hardware-based license activation and unlimited exports.

## 🚀 Tech Stack
* **Framework:** WPF / .NET (C#)
* **Architecture:** MVVM (CommunityToolkit.Mvvm)
* **Payment Integration:** TON Blockchain API
